## What It Does

qq-farm-cdp-auto is a Windows-focused automation script for QQ Farm-style mini-game tasks. It is centered on everyday gameplay actions that can be coordinated with custom CDP protocols and RPC gateways, making it a fit for users who want routine farm work handled in the background.

Built for QQ and WeChat environments, the project emphasizes hidden or minimized window operation. Its development direction is to stay aligned with those environment-specific interfaces while keeping the script package compact and practical.
